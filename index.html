My Journey

Having worked as an analyst in my previous roles, I've always found to have spent a large amount of time having to do repetetive tasks such cleaning and transforming data before I could begin running any sort of analysis. I knew these tasks could be automated but at the time I didn't have the knowledge of building such applications. I then began learning Python in order to put these ideas into practice and I did find much success in doing so. Soon after I realised, only someone familiar with Python would be able to run these scripts so how would I be able to create a system tailored for a non-technical user? I then began learning web development and creating web apps to solve business problems.
